# Tube Project Overview

## Background & Problem

* Blockchain technology is driving global financial innovation, but for most users, the entry barrier remains high.
* Engaging with DeFi, trading tokens, and utilizing NFTs often requires multiple apps, specialized wallets, complex onboarding processes, and a high level of technical knowledge.
* Blockchain-based chatbots are primarily developed for Telegram. While Telegram is not the dominant global messaging app, it has become **the leading messaging platform within the cryptocurrency market**, supported by its own Ton blockchain network, making the ecosystem increasingly robust.

## Solution

* **Tube** is a Telegram-based **agent trading bot specialized for the Sei Network**.
* Users can easily send tokens, check crypto prices, perform swaps, trade NFTs, and interact with prediction markets—all directly within the Telegram chat interface.
* **No external apps or complicated onboarding**—blockchain transactions are as simple as sending a text message.
* Powered by **AIDN-based agent automation**, Tube enables even users in emerging markets to access DeFi without prior blockchain knowledge.

## Key Differentiators

1. **Expanded Multi-Chain Support**

   * Integrates **Sei Network** into an environment that already supports Solana, Ethereum, and TON.
2. **Enhanced DeFi Accessibility in Telegram**

   * Onboarding, wallet creation, and transactions all handled within the chat interface.
3. **Sei Network-Based Marketing Capabilities**

   * Projects and communities can easily run Telegram events (airdrops, rewards, etc.) using **Sei tokens**.

## Vision

Tube aims to be **the gateway bridging communication and decentralized finance**,
serving as a core hub for expanding the Sei Network ecosystem.
From emerging markets to global adoption, Tube makes blockchain **easier, faster, and more user-friendly**.
